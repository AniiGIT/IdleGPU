Contributing
IdleGPU is currently a solo project developed by AniiGIT as part of my personal portfolio. I am not accepting general contributions or pull requests at this time.

Security Issues
Security is the exception. If you find a vulnerability, please do not open a public issue — follow the process in SECURITY.md instead. Security reports are always welcome and will be taken seriously.

Feedback
If you have ideas, suggestions, or have run IdleGPU on your own setup and want to share results, feel free to open a Discussion. I read everything even if I do not respond to every thread.

Future
Contributions may open up once the project reaches a stable foundation. If that changes, this file will be updated with full guidelines.
